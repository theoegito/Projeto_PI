# RevoluCIn - Jogo desenvolvido com Raylib
Jogo desenvolvido por Gabriela Moreira, Jefferson Perreira, Letícia Xavier, Luiz Barros, Roni Oliveira e Théo Egito.

  
## Bem-vindo a RevoluCin!
Os computadores se revoltaram! A Inteligência Artificial maligna tomou conta do CIn fazendo os computadores ganharam vida, e você deve nos ajudar a derrotá-los! Na primeira fase você deverá saltar os inimigos para ganhar acesso à uma arma que irá lhe ajudar a derrotar o chefão!
<img width="1536" height="1024" alt="home_screen" src="https://github.com/user-attachments/assets/977a743c-896f-481d-b1bf-ff2933937ea8" />


### FASE 1 
Na fase inicial trata-se de um 'runner', aonde você terá que saltar (clique 'SPACE' ou 'UP' ou 'W' para saltar) os obstáculos (computadores inimigos), ao concluir a fase você ganhará uma arma necessária para derrotar o chefão na fase 2.

### FASE 2
Nessa fase, você terá 3 vidas e usará a arma (clique 'F' ou botão esquerdo do mouse para atirar) para derrotar o chefão equanto desvia dos seus ataques.
